# Phase 5 — Аналитика по своим постам

> Личный аккаунт: counts-lite (публичные счётчики + динамика во времени). Полные insights (reach,
> impressions, saves, profile views) требуют **бизнес/проф-аккаунт** — отдельная тема с другой
> авторизацией, в фундамент НЕ входит (см. `../feasibility.md`).

## 5A — counts-lite (личный аккаунт)

Что доступно без бизнес-аккаунта: `like_count`, `comment_count`, `view_count` (видео/Reels) из `user_medias`
+ `media_likers` (кто лайкнул). Динамика — наша: периодически снимать счётчики и хранить снимки.

Объём:
- Таблица `showcase_media_metric_snapshots` (`instagram_account_id`, `media_pk`, `captured_at`,
  `like_count`, `comment_count`, `view_count`) — спроектировать при подходе. Снимки — по `is_tracked`-постам
  (из Phase 4) или по всей сетке, по расписанию (реюз scheduler автоматизации) либо по кнопке «обновить».
- Дашборд (`pages/showcase` таб «Аналитика» или отдельная страница): топ-посты, средние лайки/комменты,
  динамика по выбранному посту (sparkline/график), сводка по сетке.
- Vue: `widgets/showcase-analytics`; графики — лёгкая либа или Quasar/CSS (решить при подходе).

Чек-лист 5A:
- [ ] Таблица снимков + сбор (по кнопке и/или расписанию); агрегаты на бэке (без N+1).
- [ ] Дашборд: топы/средние/динамика; реюз TableComponent для табличной части.
- [ ] Live: счётчики совпадают с приложением; снимки копятся; динамика рисуется.

## 5B — полные insights (бизнес-аккаунт) — FUTURE

> Только если аккаунт переведён в бизнес/проф и авторизован соответствующе. Отдельный трек.

- `insights_media(media_pk)` / `insights_media_feed_all(post_type, time_frame, data_ordering, count)` —
  reach/impressions/saves/shares/profile-views и пр.
- Требует: бизнес-аккаунт + рабочая сессия с доступом к insights. Phase 0 фиксирует, что на личном они падают.
- Проектируется отдельно, когда/если появится бизнес-авторизация (она «работает по-другому» — вне текущего Android private API).

## Критерий готовности

5A: дашборд counts-lite показывает корректные данные и накапливает динамику на живом аккаунте.
5B — вне текущего объёма (бизнес-аккаунт).

## Делегирование

- 5A — **Kiro Opus (max)** (БД снимков + агрегаты бэка) + **Kiro Sonnet (max)** (дашборд FE).
- Сбор по расписанию — реюз scheduler/очереди автоматизации (швы — оркестратор).
